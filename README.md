Clone using:
`git clone --recurse-submodules git@github.com:jaimeferj/dotfiles.git`

Dependencies:
    - stow
    - zsh
    - tmux
    - fzf
    - zoxide
    - neovim


After intsalling them do:
    - On dotfiles: `stow .`
    - `chsh -s /bin/zsh`
    - Do: `tmux` and execute `ctrl+space I` to install plugins
